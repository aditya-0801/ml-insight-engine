# ML Insight Engine

A Flask-based Machine Learning application that automatically analyzes uploaded CSV datasets.

## Features

- Upload CSV datasets
- Automatic preprocessing
- Multiple ML algorithms
- Best model selection
- Accuracy comparison
- Correlation heatmap
- Confusion matrix
- Feature importance
- SQLite database
- Analysis history

## Technologies

- Python
- Flask
- Scikit-Learn
- Pandas
- Matplotlib
- Seaborn
- SQLite
